✨ 已實現的 5 大功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ 1. RGB、曝光的平均值顯示
     └─ 實時計算 R、G、B 三通道平均值 (0-255)
     └─ 顯示曝光時間 (微秒)
     └─ 自動更新統計資訊

  ✅ 2. 調整白平衡
     └─ RGB 增益滑塊控制 (0.5 ~ 2.0)
     └─ 5 種預設組合 (日光、陰天、鎢絲燈、螢光燈、陰影)
     └─ 自動白平衡 (灰卡算法)
     └─ 實時預覽效果

  ✅ 3. 顯示 HSL 色彩空間
     └─ BGR → HSV → HSL 轉換
     └─ 顯示平均 H (0-180°)、S (0-100%)、L (0-100%)
     └─ 便於色溫和飽和度調整

  ✅ 4. 顯示這張圖的平均光強
     └─ ITU-R BT.601 標準公式: Y = 0.299*R + 0.587*G + 0.114*B
     └─ 範圍: 0-255 + 百分比顯示
     └─ 實時亮度監控

  ✅ 5. 調整畫面大小
     └─ 預設縮放: 100% / 75% / 50% / 25%
     └─ 自訂寬度 (自動計算高度)
     └─ 高品質插值: linear, cubic, lanczos4
     └─ 零卡頓實時縮放

  1. 安裝依賴
     $ pip install PyQt5 opencv-python numpy psutil

  2. 執行應用
     $ python single_cam_stream.py

  3. 操作步驟
     → 點擊「🔄 刷新設備」掃描相機
     → 選擇相機後點擊「✅ 連接」
     → 點擊「▶️  開始取像」開始監控
     → 調整白平衡、縮放等參數
